# Управление автопарком

## Описание
Консольное приложение на C для управления автопарком. Позволяет вести учёт водителей, машин и заказов на перевозку грузов.

## Функционал
- Управление водителями (CRUD)
- Управление машинами (CRUD)
- Управление заказами (CRUD с проверкой грузоподъёмности)
- Автоматическое начисление 20% водителю от стоимости заказа
- Отчёты и статистика
- Автоматическое обновление статистики через триггеры SQLite

# Project Name
Autopark Management System (C)

# Description
Данный проект представляет собой консольное приложение на языке C, предназначенное для управления автопарком. Программа обеспечивает аутентификацию и авторизацию пользователей с различными ролями (администратор, водитель). Система взаимодействует с базой данных SQLite и хранит информацию о водителях, автомашинах, заказах на перевозку грузов и начислениях. Реализованы основные операции работы с данными (просмотр, добавление, обновление и удаление), автоматический расчёт 20% от стоимости заказа для водителя, проверка грузоподъёмности при добавлении заказа, а также формирование отчётов и статистики (по водителю, по машине, рейтинг водителей, рейтинг машин). Дополнительно проект покрыт автоматическими тестами.

# Installation
1. Клонировать репозиторий:
git clone https://github.com/vaannnes17/Senko-Kozlovskiy_code.git
cd Senko-Kozlovskiy_code

2. Установить зависимости:
- SQLite3
- GCC (с поддержкой C99)
- Make

На Ubuntu/Debian:
sudo apt update
sudo apt install libsqlite3-dev gcc make

На macOS:
brew install sqlite3

3. Собрать проект:
make

4. (Опционально) Запустить тесты:
make check

# Usage
1. Запустить приложение:
make run
или
./bin/autopark

2. В главном меню выбрать необходимую операцию:

| Пункт | Описание |
|-------|----------|
| 1 | Создать все таблицы |
| 2 | Добавить водителя |
| 3 | Добавить машину |
| 4 | Добавить заказ (с автоматической проверкой грузоподъёмности) |
| 5 | Просмотр всех водителей |
| 6 | Просмотр всех машин |
| 7 | Просмотр всех заказов |
| 8 | Поиск водителя по фамилии |
| 9 | Обновить стаж водителя |
| 10 | Удалить водителя |
| 11 | Удалить машину |
| 12 | Удалить заказ |
| 13 | Отчёт: заказы водителя за период |
| 14 | Отчёт: статистика по машине (общий пробег, общая масса грузов) |
| 15 | Отчёт: статистика по всем водителям (поездки, грузы, заработок) |
| 16 | Отчёт: водитель с наименьшим количеством поездок |
| 17 | Отчёт: машина с наибольшим общим пробегом |
| 0 | Выход |

3. Для проверки корректности работы:
- запускать автоматические тесты (make check)
- проверять изменения через SQLite (sqlite3 data/autopark.db)
